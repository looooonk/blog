# Taehoon Hwang Notes

This is a Next.js App Router blog that mirrors the styling of the sibling portfolio app linked at `org-source`.

## Development

```bash
npm run dev
```

Open `http://localhost:3000`.

## Content

Blog posts live in `lib/posts.ts`. Each post has a timestamp, title, tags, optional subtitle, main image, and markdown content.

The local markdown renderer supports headings, emphasis, lists, blockquotes, code fences, tables, images, links, inline math, and block math.
